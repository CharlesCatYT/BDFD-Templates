Removes all roles from a user.

Usage : ` !remove [User] `
- [] - Required

## Arguments

|       | Mention | User ID | Username |
| ----- | ------- | ------- | -------- |
| Slash |    ✓    |    ×    |    ×    |

Note : While writing the slash option name, make sure all are in lowercase format.
