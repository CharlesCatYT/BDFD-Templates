# brainfuck Compiler
Compile [brainfuck](https://esolangs.org/wiki/Brainfuck) codes.

## Notes
- Make sure you have used this command in `BDScript 2`.
- This code is slash-type command.
- This command requires 2 slash options :

    | Option | Required | Type |
    | ------------ | --------- | ----- |
    | code | True | Text |
    | input | False | Text |
