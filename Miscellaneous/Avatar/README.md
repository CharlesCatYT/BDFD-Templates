# Description
Shows author/user's default avatar.

Usage : ` !avatar {Mention/ID/Username} `

- { } - Optional

# Slash Command
Option 1

- Name : ` user `

- Type : ` User `

- Required : ` No `

# Script Language
- ` BDScript 2 `
