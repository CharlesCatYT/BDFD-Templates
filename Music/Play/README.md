# Description
Plays your requested YouTube track in your respective voice channel.

Usage : ` !play [YouTube Video Name/Link] `
* [] - Required

## Variables
Name - ` bdfd-lavalink `

Value - ` Your Host URL `

## Permission
- ` Send Message / Send Messages In Threads `
- ` Embed Links `
- ` Connect `
- ` Speak `

## Script Language
- ` BDScript 2 `
