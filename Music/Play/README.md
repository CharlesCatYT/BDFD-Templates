# Description
A music play command which let's you play the requested songs  in your respective voice channel.

# Important

## Variables

- ` Music ` - Your Lavalink

## Permission
- ` Send Message `
- ` Connect `
- ` Speak `

## Script Language
- ` BDScript 2 `
