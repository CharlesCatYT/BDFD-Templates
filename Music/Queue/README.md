# Description
Shows your current playing & all queued tracks.

Usage : ` !queue {Page Number} `
- { } - Optional

# Variables
Name - ` bdfd-lavalink `

Value - ` Your Host URL `

# Permission
- ` Send Message/Send Messages In Threads `
- ` Embed Links `

# Script Language
- ` BDScript 2 `
